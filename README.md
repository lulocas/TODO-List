
# ToDo list

## Parte em Java:
Programa simples de uma ToDo list feita em java.
Ele possui opções de adicionar, remover, listar, atualizar e consultar tarefas.

## Parte em Javascript
Possui opção de criar, excluir, listar (to do, doing, done) e atualizar as tarefas.
